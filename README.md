### Build

clone the repository,

```bash
$ git clone https://github.com/weavv/weavv-css-documentation.git
$ cd weavv-css-documentation
```

install dependencies,

```bash
# YARN
$ yarn
# NPM
$ npm i
```

serve locally,

```bash
# YARN
$ yarn dev
# NMP
$ npm run dev
```
